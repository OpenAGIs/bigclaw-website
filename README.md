# bigclaw-website

Official website starter for **bigclaw.com**.

## Upstream references

- OpenAGI.ai source: https://github.com/OpenAGIs/OpenAGI.ai
- BigClaw source: https://github.com/OpenAGIs/BigClaw

> Note: GitHub does not create a separate fork when source and destination owner are the same account.
> This repo is created as a dedicated website project, with design language aligned to OpenAGI.ai and messaging aligned to BigClaw.

## Design intent

- 大气简洁（深色、留白、信息层级清晰）
- 体现 BigClaw / OpenClaw 的核心价值：
  - 分布式并行执行
  - 可编排工作流
  - 可观测、可审计、可回放

## Local preview

Open `index.html` directly in browser, or run any static server:

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Structure

- `index.html` – landing page
- `styles.css` – visual style
