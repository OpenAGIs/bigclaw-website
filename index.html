BigClaw 特征能力

Scheduler + Priority Queue

按优先级、风险与资源约束进行并行调度，避免任务拥塞与盲目重试。

Workflow DSL

将任务目标、工具权限、验收标准、升级策略编译成可复用执行蓝图。

Runtime Sandbox

在隔离执行环境运行 Agent 任务，限制危险调用并保留完整执行上下文。

Triage & Human Takeover

对失败、阻塞、待审批任务自动分流，必要时快速切换人工接管。

Replay & Benchmark

回放历史执行并对比策略版本，量化性能、质量与稳定性变化。

Observability Ledger

统一记录 logs / traces / artifacts / audit events，支持审计与复盘报告。
